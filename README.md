# Planet AR Demo (Unity)

A simple Augmented Reality demo built with Unity that allows users to select
different planets using UI buttons and spawn them on any detected real-world
surface in AR.

## Features
- UI buttons for planet selection
- Tap to place the selected planet on detected AR surfaces
- Real-world surface (plane) detection
- Runtime spawning of 3D planet models
- Simple and clean UI flow

## Tech Stack
- Unity
- C#
- AR Foundation
- ARCore / ARKit
- Unity Canvas UI

## What This Project Demonstrates
- AR plane detection and object placement
- UI-driven object selection
- Runtime instantiation of 3D objects in AR
- Basic AR interaction workflow

## Platform
- Android (ARCore supported devices)

## How to Run
1. Open the project in Unity
2. Switch the build platform to Android
3. Ensure AR Foundation and ARCore/ARKit packages are installed
4. Build and run on a supported device
5. Select a planet from the UI and tap on a surface to place it

## Note
This project is intended as a demo to showcase AR interaction and UI-based
object selection. Planet models are used for demonstration purposes only.
